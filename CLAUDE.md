# Tienda de Tenis — Contexto del Proyecto

## ¿Qué es este proyecto?
Tienda virtual de artículos de tenis construida con React y TypeScript.
Proyecto de aprendizaje para practicar desarrollo web moderno.

## Stack tecnológico
- React con TypeScript
- create-react-app

## Estructura del proyecto
- `src/` — código fuente principal
- `public/` — archivos estáticos

## Convenciones
- Nombres de componentes en PascalCase (ej: `ProductCard.tsx`)
- Nombres de archivos en kebab-case (ej: `product-card.tsx`)
- Comentarios en español
- Un componente por archivo

## Lo que vende la tienda
- Raquetas
- Ropa deportiva
- Calzado
- Accesorios (pelotas, overgrips, bolsos)
